Windows Messenger-client

    Met de client-toepassing Windows CE Messenger in Windows CE kunt u in real-time communiceren met behulp van het berichtensysteem, VoIP en meer.

    • Zie wie er on line is.
      Zie wanneer uw collega's on line zijn. U kunt ook uw eigen aanwezigheid bekend maken aan een lijst met mensen die u bepaalt.
    • Praten in plaats van typen.
      Stop met typen! Praat met een collega waar dan ook ter wereld met gebruik van de microfoon en luidspreker op uw apparaat. Windows Messenger ondersteunt voice calling vanaf uw Windows CE-apparaat.
    • Communiceer met gebruik van SIP (Session Initiation Protocol) of Hotmail-contacten.
      Windows Messenger ondersteunt communicatie met uw collega's in uw onderneming via een communicatieservice met SIP. U kunt ook praten met Hotmail-contacten via .NET Messenger.
