Client Windows Messenger

    L'applicazione client Windows CE Messenger di Windows CE consente di comunicare in tempo reale mediante messaggi di testo, VoIP e altri metodi.

    • Individuare chi è in linea.
      È possibile vedere quando i colleghi sono in linea. Inoltre, è possibile inserire le informazioni relative aloppuropria presenza in un elenco di analizzatori definito personalmente.
    • Parlare invece di utilizzare la tastiera
      Invece di utilizzare la tastiera per comunicare, è possibile parlare direttamente con un collega ovunque ci si trovi mediante il microfono e gli altoparlanti del dispositivo. Windows Messenger supporta le funzioni di chiamata vocale di alta qualità disponibili nel dispositivo Windows CE.
    • Comunicare mediante il protocollo SIP (Session Initiation Protocol) oppure i contatti Hotmail
      Windows Messenger consente di comunicare con i colleghi di lavoro mediante un servizio di comunicazione basato sul protocollo SIP. Utilizzando .NET Messenger, è inoltre possibile comunicare con i propri contatti Hotmail.
