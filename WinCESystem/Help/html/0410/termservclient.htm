Informazioni su Connessione desktop remoto

Tramite Connessione desktop remoto, è possibile accedere a un server terminal e utilizzare tutti i programmi che vi sono installati. Ad esempio, invece di eseguire Microsoft Pocket Word, è possibile eseguire la versione per PC di Microsoft Word.

    Connessione a un server terminal
    Suggerimenti e consigli

    Connessione a un server terminal
    1. Selezionare Start > Programmi > Connessione desktop remoto oppure eseguire il comando "CETSC" dal prompt dei comandi.
    2. Nella casella Server, digitare o selezionare il nome del server terminal o l'indirizzo TCP/IP, oppure selezionare un server nell'elenco a discesa.
    3. Scegliere il pulsante Connetti.
    4. Nella finestra Connessione desktop remoto, digitare il nome utente, la password e, se richiesto, il nome del dominio, quindi scegliere OK.

      Disconnessione lasciando la sessione aperta
      1. Nella finestra Connessione desktop remoto, selezionare Start > Chiudi sessione.
      2. Selezionare Disconnetti > OK.

      Nota  Se in precedenza ci si è disconnessi da un server terminal senza chiudere la sessione, il server terminal continuerà a eseguire i processi in corso e tramite Connessione desktop remoto sarà possibile connettersi nuovamente alla stessa sessione (a condizione che l'amministratore abbia configurato Connessione desktop remoto per la nuova connessione alle sessioni disconnesse).

          Ottimizzazione delle prestazioni video
          È possibile ridurre la frequenza di aggiornamento dello schermo.
          1. Nella finestra del server terminal, selezionare Start > Impostazioni > Pannello di controllo.
          2. Selezionare Schermo.  
          3. Nelle versioni più recenti dei server terminal, potrebbe dover essere necessario selezionare Effetti nella scheda Aspetto.  
          4. Disattivare le opzioni Utilizza effetti per menu e descrizione comandi e Smussa gli angoli dei caratteri dello schermo.

            Suggerimenti e consigli

            Ottimizzazione delle prestazioni di Internet Explorer
            È possibile ridurre la frequenza di aggiornamento delle pagine Web.
            1. In Internet Explorer, selezionare Strumenti > Opzioni Internet.
            2. Selezionare la scheda Avanzate.
            3. Rimuovere il segno di spunta dalla casella Usa scorrimento continuo.
