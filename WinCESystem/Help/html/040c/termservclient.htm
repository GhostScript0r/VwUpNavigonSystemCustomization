À propos de la Connexion Bureau à distance

Grâce la Connexion Bureau à distance, vous pouvez vous connecter à un serveur Windows Terminal Server et utiliser tous les programmes installés sur ce serveur. Par exemple, au lieu d'exécuter Microsoft Pocket Word, vous pouvez exécuter la version bureau de Microsoft Word.

    Connexion à un Terminal Server
    1. Sélectionnez Démarrer > Programmes > Connexion Bureau à distance, ou exécutez « CETSC » à partir de l'invite de la commande.
    2. Dans la boîte Serveur, tapez ou sélectionnez un nom de Terminal Server ou une adresse TCP/IP, ou bien sélectionnez un serveur dans la zone de liste déroulante.
    3. Sélectionnez Se connecter.
    4. Dans la fenêtre Connexion Bureau à distance, entrez vos nom d'utilisateur, mot de passe et domaine (le cas échéant), puis cliquez sur OK.

      Se déconnecter sans fermer la session
      1. Dans la fenêtre Connexion Bureau à distance, sélectionnez Démarrer > Arrêt.
      2. Sélectionnez Déconnecter > OK.

      Remarque :  si vous vous êtes précédemment déconnecté d'un Terminal Server sans arrêter la session, le Terminal Server poursuivra l'exécution de tout processus en cours et Connexion Bureau à distance pourra ultérieurement se reconnecter à la même session (pour autant que votre administrateur ait configuré Connexion Bureau à distance pour qu'elle se reconnecte aux sessions interrompues).

          Amélioration des performances d'affichage
          Vous pouvez réduire la durée de rafraîchissement de l'écran.
          1. Dans la fenêtre Terminal Server, sélectionnez Démarrer > Paramètres > Panneau de configuration.
          2. Sélectionnez Affichage.  
          3. Sur certains serveurs Terminal Server récents, vous devrez peut-être sélectionner Effets dans l'onglet Apparence.  
          4. Désactivez Effets de transition pour les menus et les info-bulles et Lisser les polices d'écran.

            Amélioration des performances avec Internet Explorer
            Vous pouvez réduire la durée de rafraîchissement des pages Web.
            1. Dans Internet Explorer, sélectionnez Outils > Options Internet.
            2. Sélectionnez l'onglet Avancé.
            3. Désactivez Utiliser le défilement régulier.
