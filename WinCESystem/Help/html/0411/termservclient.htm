リモート デスクトップ接続について

リモート デスクトップ接続を使用して Windows ターミナル サーバーにログ オンし、Windows ターミナル サーバーにインストールされているすべてのプログラムを使用することができます。例えば、Microsoft Pocket Word を実行する代わりに、デスクトップ バージョンの Microsoft Word を実行することができます。

    関連項目
    ターミナル サーバーへの接続
    ヒント

    ターミナル サーバーへの接続
    1. [スタート] を選択し、[プログラム] をポイントして、[リモート デスクトップ接続] を選択します。またはコマンド プロンプトから 'CETSC' を実行します。
    2. [サーバー] ボックスにターミナル サーバー名または TCP/IP アドレスを入力するか、ドロップ ダウン リストからサーバーを選択します。
    3. [接続] をタップまたはクリックします。
    4. [リモート デスクトップ接続] ウィンドウ内で、ユーザー名、パスワード、そしてドメイン (必要な場合) を入力して [OK] をクリックします。

      セッションを終了せずに切断
      1. [リモート デスクトップ接続] ウィンドウ内で、[スタート] ボタンを選択し、[シャットダウン] を選択します。
      2. [切断] を選択し、[OK] をタップまたはクリックします。

      メモ  セッションを終了せずにターミナル サーバーから切断した場合、ターミナル サーバーは実行中のプロセスを実行し続け、リモート デスクトップ接続は後からこの同じセッションに再接続することが可能です (管理者がリモート デスクトップ接続が切断したセッションに再接続できるように構成した場合)。

        切断してセッションを終了
        ヒント

        切断してセッションを終了
        1. [リモート デスクトップ接続] ウィンドウ内の [スタート] を選択し、[シャットダウン] を選択します。
        2. [ログオフ] を選択し、[OK] をタップまたはクリックします。

          ヒント
          リモート デスクトップ接続を使用する際に役に立つヒントです。

          画面のパフォーマンス向上
          Internet Explorer でのパフォーマンス向上
          CETSC のコマンド ライン オプション


          画面のパフォーマンス向上
          画面が再描画される時間を縮減することができます。
          1. [ターミナル サーバー] ウィンドウ内の [スタート] を選択します。[設定] を選択し、[コントロール パネル] をクリックします。
          2. [画面] を選択します。  
          3. 一部の新しいターミナル サーバーでは、[デザイン] タブ内の [効果] を選択する必要がある場合があります。  
          4. [メニューとツールヒントの切り替え効果] および [スクリーン フォントの縁を滑らかにする] チェック ボックスをオフにします。
            関連項目
            ヒント

            Internet Explorer でのパフォーマンス向上
            Web ページが再描画される時間を縮減することができます。
            1. Internet Explorer から [ツール] をポイントし、[インターネット オプション] を選択します。
            2. [詳細設定] タブを選択します。
            3. [スムーズ スクロールを使用する] チェック ボックスをオフにします。
              関連項目
              ヒント

              CETSC コマンド ライン オプション
              'CETSC /?' コマンドでビルト イン ヘルプを表示し、追加のコマンド ライン オプションを参照することができます。
