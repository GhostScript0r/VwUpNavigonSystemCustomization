Windows Messenger クライアント

    Windows CE 上の Windows CE Messenger クライアント アプリケーションでは、テキストベースのメッセージング、VoIP、およびその他を使用してリアルタイムでの通信を可能にします。

    • オンラインのメンバを表示する
      同僚がオンラインしたとき表示します。定義した公開先メンバへオンラインの状態を知らせることができます。
    • タイプせずに話す
      タイプすることをやめましょう。デバイス上のマイクおよびスピーカを使用して世界中の同僚たちと会話ができます。Windows Messenger では Windows CE デバイスからの高質の音声チャットが可能です。
    • Session Initiation Protocol (SIP) または Hotmail の連絡先を使用しての通信
      Windows Messenger では SIP ベースの通信サービスを使用している会社にいる同僚との通信をサポートします。また、.NET Messenger を使用して Hotmail の連絡先と会話をすることが可能です。
