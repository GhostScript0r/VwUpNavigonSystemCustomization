원격 데스크톱 연결 정보

원격 데스크톱 연결을 사용하여 Windows Terminal Server에 로그온한 다음 이 서버에 설치되어 있는 모든 프로그램을 사용할 수 있습니다. 예를 들어, Microsoft Pocket Word 대신 Microsoft Word 데스크톱 버전을 실행할 수 있습니다.

    참고 항목

    유용한 정보

    터미널 서버에 연결
    1. 시작 > 프로그램 > 원격 데스크톱 연결을 선택하거나 명령 프롬프트에서 'CETSC'를 실행합니다.
    2. 서버 상자에서 터미널 서버 이름 또는 TCP/IP 주소를 입력 또는 선택하거나, 드롭다운 목록에서 서버를 선택합니다.
    3. 연결을 선택합니다.
    4. 원격 데스크톱 연결 창에서 사용자 이름, 암호, 도메인(필요한 경우)을 입력한 다음 확인을 선택합니다.

      세션을 끝내지 않고 연결 끊기
      1. 원격 데스크톱 연결 창에서 시작 > 시스템 종료를 선택합니다.
      2. 연결 끊기 > 확인을 선택합니다.

      참고  이전에 세션을 끝내지 않고 터미널 서버 연결을 해제한 경우 터미널 서버가 실행 중인 프로세스를 계속해서 실행할 것이므로, 원격 데스크톱 연결을 통해 나중에 이 세션에 다시 연결할 수 있습니다(관리자가 연결이 끊어진 세션에 다시 연결하도록 원격 데스크톱 연결을 설정한 경우).

          디스플레이 성능 향상
          화면을 다시 표시하는 데 걸리는 시간을 줄일 수 있습니다.
          1. 터미널 서버 창에서 시작 > 설정 > 제어판을 선택합니다.
          2. 디스플레이를 선택합니다.  
          3. 일부 최신 터미널 서버에서는 화면 배색 탭에서 효과를 선택해야 할 수도 있습니다.   
          4. 메뉴 및 도구 설명에 전환 효과 사용과 화면 글꼴의 가장자리 다듬기의 선택을 취소합니다.

            Internet Explorer 사용 시 성능 향상
            웹 페이지를 다시 표시하는 데 걸리는 시간을 줄일 수 있습니다.
            1. Internet Explorer에서 도구 > 인터넷 옵션을 선택합니다.
            2. 고급 탭을 선택합니다.
            3. 부드러운 화면 이동의 선택을 취소합니다.
