Windows Messenger 클라이언트

    Windows CE 의 Windows CE Messenger 클라이언트 응용 프로그램을 사용하면 텍스트 기반 메시지 보내기, VoIP 등을 사용하여 실시간으로 통신할 수 있습니다.

    • 온라인상에 있는 사용자 표시
      동료가 온라인상에 있는지 알 수 있습니다. 지정한 사용자 목록에만 자신의 온라인 상태 정보가 표시되도록 할 수도 있습니다.
    • 타이핑이 아닌 대화
      타이핑은 그만! 장치에 있는 마이크와 스피커를 사용하여 전세계 어느 곳에 있는 동료와도 얘기를 나눌 수 있습니다. Windows Messenger는 Windows CE 장치를 통해 고음질의 음성 채팅 기능을 지원합니다.
    • SIP(Session Initiation Protocol) 또는 Hotmail 연락처를 사용한 통신
      Windows Messenger는 SIP 기반 통신 서비스를 사용하여 회사 동료와 통신하는 기능을 지원합니다. 또한 .NET Messenger를 사용하여 Hotmail 사용자와 얘기를 나눌 수도 있습니다.
