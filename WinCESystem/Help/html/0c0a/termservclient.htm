Acerca de Conexión a Escritorio remoto

Gracias a Conexión a Escritorio remoto, puede iniciar una sesión de Windows Terminal Server y utilizar todos los programas instalados en el servidor. Por ejemplo, en lugar de utilizar Microsoft Pocket Word, podrá ejecutar la versión de escritorio de Microsoft Word.

    Conexión con Terminal Server
    1. Seleccione Inicio > Programas > Conexión a Escritorio remoto o ejecute "CETSC" desde el símbolo del sistema.
    2. En el cuadro Servidor, escriba o seleccione un nombre de Terminal Server o una dirección TCP/IP, o seleccione un servidor de la lista desplegable.
    3. Seleccione Conectar.
    4. En la ventana Conexión a Escritorio remoto, escriba el nombre de usuario, la contraseña y el dominio (si es necesario) y, a continuación, seleccione Aceptar.

      Desconexión sin finalizar una sesión
      1. En la ventana Conexión a Escritorio remoto, seleccione Inicio > Apagar.
      2. Seleccione Desconectar > Aceptar.

      Nota  Si se desconectó de Terminal Server sin cerrar anteriormente la sesión, éste continuará ejecutando los procesos abiertos y Conexión a Escritorio remoto podrá volver a conectarse a esta misma sesión (sólo si su administrador configuró Conexión a Escritorio remoto para volver a conectarse a las sesiones desconectadas).

          Mejora de las opciones de presentación
          Puede disminuir el intervalo de tiempo necesario para actualizar la pantalla.
          1. En la ventana del cliente de Terminal Server, seleccione Inicio > Configuración > Panel de control.
          2. Seleccione Pantalla.  
          3. En algunos de los Terminal Server más recientes, es probable que también tenga que seleccionar Efectos en la ficha Apariencia.  
          4. Desactivar Aplicar animaciones a los menús e información sobre herramientas y Alisar bordes para las fuentes de pantalla.

            Sugerencias y trucos

            Mejora del rendimiento con Internet Explorer
            Puede disminuir el intervalo de tiempo necesario para actualizar páginas Web.
            1. En Internet Explorer, seleccione Herramientas > Opciones de Internet.
            2. Seleccione la ficha Opciones avanzadas.
            3. Desactivar Usar desplazamiento suave.

              Opciones de línea de comandos CETSC
              Para utilizar opciones de línea de comandos adicionales, escriba: CETSC /? en la ayuda integrada.
