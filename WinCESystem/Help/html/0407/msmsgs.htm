Windows Messenger-Client

    Die Windows CE Messenger-Clientanwendung in Windows CE ermöglicht eine Echtzeit-Kommunikation über text-basierte Nachrichten, VoIP und vieles mehr.

    • Sehen, wer online ist.
      Sehen, wenn Ihre Kollegen online sind. Sie können Ihre eigenen Anwesenheitsinformationen ebenfalls zu einer von Ihnen definierten Watcher-Liste hinzufügen.
    • Reden anstatt zu tippen
      Sparen Sie sich das Tippen! Mithilfe von Mikrofon und Lautsprecher können Sie sich mit Ihren Kollegen aus der ganzen Welt unterhalten. Windows Messenger unterstützt modernstes Voice Calling auf Ihrem Gerät.
    • Kommunizieren Sie mithilfe von Session Initiation Protocol (SIP)- oder Hotmail-Kontakten
      Windows Messenger unterstützt die Kommunikation mit Ihren Kollegen innerhalb der Firma durch einen SIP-basierten Kommunikationsdienst. Darüber hinaus können Sie auch mit Hilfe von .NET Messenger mit Ihren Hotmail-Kontakten sprechen.
