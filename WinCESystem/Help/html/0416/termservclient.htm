Sobre a Conexão da Área de Trabalho Remota

Com Conexão da Área de Trabalho Remota, você pode fazer logon em um Windows Terminal Server e usar todos os programas instalados neste servidor. Por exemplo, em vez de executar o Microsoft Pocket Word, você pode executar a versão para o computador de mesa do Microsoft Word.

    Estabelecendo Conexão com um Terminal Server
    1. Selecione Iniciar > Programas > Conexão de Área de Trabalho Remota ou execute 'CETSC' a partir do prompt de comando.
    2. Na caixa Servidor, digite ou selecione um nome de Terminal Server ou endereço TCP/IP, ou selecione um servidor na lista suspensa.
    3. Selecione Conectar.
    4. Na janela Conexão da Área de Trabalho Remota, digite seu nome de usuário, senha e domínio (se necessário) e selecione OK.

      Desconectando sem Encerrar uma Sessão
      1. Na janela Conexão da Área de Trabalho Remota, selecione Iniciar > Desligar.
      2. Selecione Desconectar > OK.

      Observação  Se você já tiver desconectado de um Terminal Server sem encerrar a sessão, o Terminal Server continuará executando todos os processos em andamento e a Conexão da Área de Trabalho Remota poderá reconectar à mesma sessão mais tarde (se o seu administrador tiver configurado a Conexão da Área de Trabalho Remota para se reconectar a sessões desconectadas).

          Aprimorando o desempenho da exibição
          É possível reduzir o tempo necessário para redesenhar a tela.
          1. Na janela Terminal Server, selecione Iniciar > Configurações > Painel de Controle.
          2. Selecione Exibir.  
          3. Em alguns Terminal Servers novos, é possível que você também tenha que selecionar Efeitos na guia Aparência.  
          4. Limpar Efeitos de transição para menus e dicas de ferramentas e Usar fontes da tela com cantos arredondados.

            Aprimorando o desempenho com o Internet Explorer
            É possível reduzir o tempo necessário para redesenhar páginas da Web.
            1. No Internet Explorer, selecione Ferramentas > Opções da Internet.
            2. Selecione a guia Avançado.
            3. Limpe Use rolagem suave.
