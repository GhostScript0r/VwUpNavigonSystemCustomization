关于远程桌面连接

使用远程桌面连接，您可以登录到 Windows 终端服务器并使用该服务器上安装的所有程序。例如，您可以不用运行 Microsoft Pocket Word，而代之以 Microsoft Word 的桌面版本。

    连接到终端服务器
    1. 选择“开始”>“程序”>“远程桌面连接”，或在命令提示符下运行“CETSC”。
    2. 在“服务器”框中，键入或选择终端服务器名称或 TCP/IP 地址，或从下拉列表中选择一个服务器。
    3. 选择“连接”。
    4. 在“远程桌面连接”窗口中，键入您的用户名、密码和域名（如果必需），然后选择“确定”。

      断开连接但不结束会话
      1. 在“远程桌面连接”窗口中，选择“开始”>“关机”。
      2. 选择“断开连接”>“确定”。

      注意：如果您已从终端服务器断开连接但未结束会话，终端服务器将继续执行处于运行中的进程，远程桌面连接以后可以再连接到该会话（如果您的管理员已为重新连接到断开的会话配置了远程桌面连接）。

        断开连接并结束会话
        1. 在“远程桌面连接”窗口中，选择“开始”>“关机”。
        2. 选择“注销”>“确定”。

          改善显示性能
          您可以减少屏幕重绘时间。
          1. 在“终端服务器”窗口中，选择“开始”>“设置”>“控制面板”。
          2. 选择“显示”。
          3. 如果是较新的终端服务器，您可能还需要从“外观”选项卡中选择“效果”。
          4. 清除“动画显示菜单和工具提示”及“平滑屏幕字体边缘”。

            提示与窍门

            改善 Internet Explorer 性能
            您可以减少 Web 页的重绘时间。
            1. 在 Internet Explorer 中，选择“工具” >“Internet 选项”。
            2. 选择“高级”选项卡。
            3. 清除“使用平滑滚动”。
