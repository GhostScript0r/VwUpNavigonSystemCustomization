Windows Messenger 客户端

    Windows CE 中的 Windows CE Messenger 客户端应用程序允许用户使用基于文本的消息、VoIP 等进行实时通信。

    • 查看处在联机状态的用户。
      查看同事处在联机状态的时间。用户还可以将自己已处于联机的信息发送到由自己定义的一列观察程序。
    • 进行交谈而非键入信息
      别再键入信息了！使用设备中的麦克风和扬声器与世界各地的同事进行交谈。Windows Messenger 支持来自 Windows CE 设备的高质量语音呼叫。
    • 使用 Session Initiation Protocol (SIP) 或 Hotmail 联系人进行通信。
      Windows Messenger 提供基于 SIP 的通信服务，支持与企业中的同事进行通信。用户还可以与使用 .NET Messenger 的 Hotmail 联系人进行交谈。
