    辅助功能

      使用“控制面板”中的“辅助功能”可自定义键盘、显示或鼠标的工作方式。

      “控制面板”中的“辅助功能”工具可实现各种各样的功能。

      粘滞键
      切换键
      声音卫士
      高对比度
      鼠标键

      注意：要打开辅助功能，请选择“开始”>“设置”>“控制面板”，然后双击“辅助功能”。


        粘滞键

          使用粘滞键可通过一次按键替代同时按下多键。

          设计粘滞键的目的是给无法同时按下多键的人带来方便。如果快捷方式要求使用组合键（如 CTRL+P），可通过粘滞键使按下的修改键（CTRL、ALT 或 SHIFT）或 Windows 徽标键一直保持有效，直至按下另一键。

          要启用粘滞键，请执行以下操作：

          1. 打开辅助功能：选择“开始”>“设置”>“控制面板”，双击“辅助功能”。
          2. 在“键盘”选项卡的“粘滞键”下方，选中“使用粘滞键”复选框。

          要关闭粘滞键，请清除“使用粘滞键”复选框。

          注意：如果选中“粘滞键设置”对话框中的“使用快捷键”复选框，您可通过按五次 SHIFT 键来启用或关闭粘滞键。

            更改粘滞键选项

              设计粘滞键的目的是给无法同时按下多键的人带来方便。如果快捷方式要求使用组合键（如 CTRL+P），可通过粘滞键使按下的修改键（CTRL、ALT 或 SHIFT）或 Windows 徽标键一直保持有效，直至按下另一键。

              1. 打开辅助功能：选择“开始”>“设置”>“控制面板”，双击“辅助功能”。
              2. 在“键盘”选项卡的“粘滞键”下方，单击“设置”。执行下列操作之一：
                • 要使用允许通过按五次 Shift 键来打开或关闭粘滞键的快捷方式，请选中“使用快捷键”复选框。
                • 要指定连续按两次修改键（Ctrl、Alt 或 Shift）或 Windows 徽标键并使其保持激活状态（直至按下另一键），请选中“选项和通知”下方的“按两次修改键锁定”复选框。
                • 要在按下、锁定或释放修改键（Ctrl、Alt 或 Shift）或 Windows 徽标键时发出声音提示，请选中“选项和通知”下方的“按下修改键时发出声音”复选框。

              切换键

                切换键会在用户按下某些锁定键时发出声音。

                设计切换键的目的是考虑有视觉或认知障碍的人的需要。一旦启用切换键，计算机将在您按下锁定键（CAPS LOCK、NUM LOCK 或 SCROLL LOCK）时发出声音提示。打开锁定键的提示音较大，关闭锁定键的提示音较小。

                要启用切换键，请执行以下操作：

                1. 打开辅助功能：选择“开始”>“设置”>“控制面板”，双击“辅助功能”。
                2. 在“键盘”选项卡的“切换键”下方，选中“使用切换键”复选框。

                要关闭切换键，请清除“使用切换键”复选框。

                注意：
                • 要使用允许通过按住 NUM LOCK 键八秒钟来启用或关闭切换键的键盘快捷方式，请在“键盘”选项卡的“切换键”下方单击“设置”。
                • 如果选中“切换键设置”对话框中的“使用快捷键”复选框，您可通过按住 NUM LOCK 键八秒钟来启用或关闭切换键。

                声音卫士

                  声音卫士提供了系统声音的视觉警告。

                  设计声音卫士的目的是方便有听力障碍的人使用计算机。通过声音卫士，您可更改一些设置来生成视觉警告。例如，在计算机发出声音时显示闪烁的标题栏或边框。

                  要启用声音卫士，请执行以下操作：

                  1. 打开辅助功能：选择“开始”>“设置”>“控制面板”，双击“辅助功能”。
                  2. 在“声音”选项卡的“声音卫士”下方，选中“使用声音卫士”复选框。

                  要关闭声音卫士，请清除“使用声音卫士”复选框。


                  高对比度

                    高对比度可通过备用颜色和字号来提高屏幕的对比度。

                    设计高对比度的目的是方便有视觉障碍的人。使用高对比度颜色方案的备用颜色组合，屏幕对比度可得到增强，屏幕因此更易于观看。有些方案还采用更改字号来方便用户阅读。

                    要启用高对比度，请执行以下操作：

                    1. 打开辅助功能：选择“开始”>“设置”>“控制面板”，双击“辅助功能”。
                    2. 在“显示”选项卡的“高对比度”下方，选中“使用高对比度”复选框。

                    要关闭高对比度，请清除“使用高对比度”复选框。

                    注意：如果选中“高对比度设置”对话框中的“使用快捷键”复选框，您可通过按下左 ALT + 左 SHIFT + PRINT SCREEN 键（取决于“辅助功能”对话框中的其他设置）来启用或关闭高对比度。要打开“高对比度设置”对话框，请打开“辅助功能”，选择“显示”选项卡，然后在“高对比度”下方单击“设置”。

                      更改高对比度选项

                        设计高对比度的目的是方便有视觉障碍的人。使用高对比度颜色方案的备用颜色组合，屏幕对比度可得到增强，屏幕因此更易于观看。有些方案还可更改字号来方便用户阅读。

                        1. 打开辅助功能：选择“开始”>“设置”>“控制面板”，双击“辅助功能”。
                        2. 在“显示”选项卡的“高对比度”下方，单击“设置”。执行下列操作之一：
                          • 要使用允许通过按左 Alt + 左 Shift + Print Screen 键来启用或关闭高对比度的快捷方式，请选中“使用快捷键”复选框。
                          • 要更改高对比度的外观方案，请选中“高对比度配色方案”中的选项。

                        鼠标键

                          鼠标键允许由键盘执行鼠标功能。

                          设计鼠标键的目的是方便使用鼠标有困难的人。通过鼠标键，您可使用数字键盘来控制鼠标指针。如果要使用数字键盘输入数据或导航，可按下 NUM LOCK 键激活鼠标键。

                          1. 打开辅助功能：选择“开始”>“设置”>“控制面板”，双击“辅助功能”。
                          2. 在“鼠标”选项卡的“鼠标键”下方，单击“设置”。
                          3. 根据需要完成下列操作：
                            • “使用快捷键”复选框
                              选中该复选框，您可使用通过按左 Alt + 左 Shift + Num Lock 键来启用或关闭鼠标键的快捷方式。
                            • “最高速度”滑块
                              拖动该滑块可设置按住“鼠标键”方向键时鼠标指针移动的最大速度。
                            • “加速”滑块
                              拖动该滑块可设置按住“鼠标键”方向键时鼠标指针达到最大速度的加速度。
                            • “Ctrl 键加速，Shift 键减速”复选框
                              选中该复选框，您可在按住“鼠标键”方向键时，按 Shift 键减慢指针速度，按 Ctrl 键加快指针速度。
                            • “使用鼠标键，此时 NumLock 键为”
                              “关闭” -- 如果使用键盘输入数据，选中“关闭”可通过按 NumLock 键在鼠标键和标准数字键操作间切换。
                              “打开” -- 如果主要使用数字键盘进行移动操作，请选中“打开”。

                          启用鼠标键

                            设计鼠标键的目的是方便使用鼠标有困难的人。通过鼠标键，您可使用数字键盘来控制鼠标指针。如果要使用数字键盘输入数据或导航，可按下 NUM LOCK 键激活鼠标键。

                            要启用鼠标键，请执行以下操作：

                            1. 打开辅助功能。
                            2. 在“鼠标”选项卡的“鼠标键”下方，选中“使用鼠标键”复选框。

                            要关闭“鼠标键”，请清除“使用鼠标键”复选框。

                            另请参阅
                            使用击键启用鼠标键

                            使用击键启用鼠标键

                              1. 按左 Alt + 左 Shift + NUM LOCK 键。
                              2. 在鼠标键信息对话框中，按空格键清除“为辅助功能特性关闭键盘快捷方式”复选框。
                              3. 按 Tab 键，然后按 Enter 键。

                              如果使用上述击键未打开鼠标键，请按照以下步骤操作：

                              1. 打开辅助功能。
                              2. 按 Ctrl + Tab 键，直到选中“鼠标”选项卡。
                              3. 要选中“使用鼠标键”，请按空格键。
                              4. 按 Tab 键两次选中“确定”，然后按 Enter 键。
                              注意：
                              • 要更改鼠标键设置，请在“鼠标”选项卡中单击“设置”。
                              • 如果选中“鼠标键设置”对话框中的“使用快捷键”复选框，您可通过按左 Alt + 左 Shift + NUM LOCK 键来打开或关闭鼠标键。

                              使用鼠标键单击

                                1. 如果尚未这样做，请启用鼠标键。要启用鼠标键，请打开辅助功能，选择“鼠标”选项卡，然后选中“使用鼠标键”复选框。
                                2. 使用数字键盘中的按钮执行单击、双击或右键单击操作。如果鼠标键不起作用，请按 NUM LOCK 键，然后重试。

                                  要点 ！！ 请在数字键盘中执行所有鼠标键操作。

                                • 左击
                                    要进行左单击，请按除号键 (/) 激活左方向，然后按 5 单击。
                                  • 要进行左双击，请按除号键 (/) 激活左方向，然后按加号 (+) 双击。
                                • 右击
                                  • 要进行右单击，请按减号 (-) 激活右方向，然后按 5 单击。
                                  • 要进行右双击，请按减号 (-) 激活右方向，然后按加号 (+) 双击。
                                注意：
                                • 要模拟同时使用两个鼠标按钮的情形，请按数字键盘上的星号 (*)，然后按 5 进行单击，或按加号 (+) 进行双击。
                                • 要切换回标准击键，请按数字键盘上的斜线号 (/)。
                                • 要指定使用鼠标键时 NUM LOCK 键是否必须处于打开或关闭状态，请在“鼠标”选项卡中单击“设置”，然后在“使用鼠标键，此时 NumLock 键为”中单击“打开”或“关闭”。
                                • 要更改使用鼠标键时指针的速度和加速设置，请在“鼠标”选项卡中单击“设置”，然后移动滑块来调整指针的速度和加速设置。
                                • 要临时加快或减慢指针速度，请在“鼠标”选项卡中单击“设置”，然后选中“Ctrl 键加速，Shift 键减速”复选框。

                                使用鼠标键拖动

                                  1. 如果尚未这样做，请启用鼠标键。要启用鼠标键，请打开辅助功能，选择“鼠标”选项卡，然后选中“使用鼠标键”复选框。
                                  2. 使用数字键盘中的按钮执行拖放对象的操作。如果鼠标键不起作用，请按 NUM LOCK 键，然后重试。
                                    要点 ！！ 请在数字键盘中执行所有鼠标操作。

                                  • 要使指针在要移动或拖动到新位置的对象上移动，请按箭头键（2、4、6 和 8）。
                                  • 要左移指针，请按除号 (/) 激活方向，然后按所需的箭头键。
                                  • 要右移指针，请按减号 (-) 激活方向，然后按所需的箭头键。
                                  • 要按住鼠标按钮，请按数字键盘上的 Ins 键 (0)。
                                  • 要释放鼠标按钮，请按数字键盘上的 DEL 键 (.)。

                                    例如，要选择对象并拖至左侧，请按如下顺序按键：

                                    /（激活左方向）INS（下移）<- <- <-（左移）DEL（上移）
                                  注意：
                                  • 要指定使用鼠标键时 NUM LOCK 键是否必须处于打开或关闭状态，请在“鼠标”选项卡中单击“设置”，然后在“使用鼠标键，此时 NumLock 键为”中单击“打开”或“关闭”。
                                  • 要更改使用鼠标键时指针的速度和加速设置，请在“鼠标”选项卡中单击“设置”，然后移动滑块来调整指针的速度和加速设置。
                                  • 要临时加快或减慢指针速度，请在“鼠标”选项卡中单击“设置”，然后选中“Ctrl 键加速，Shift 键减速”复选框。

                                        设置日期和时间

                                          1. 选择“开始”>“设置”>“控制面板”。
                                          2. 打开“日期/时间”图标。

                                                设置设备管理

                                                第一次设置“设备管理”程序时，您将连接到设备管理服务器来下载软件包并安装在设备中。

                                                1. 选择“开始” >“设置” >“控制面板”。
                                                2. 单击“设备管理”图标。
                                                3. 在“设备管理”对话框中，键入设备管理服务器的名称或 IP 地址。如果不清楚设备管理服务器的名称或 IP 地址，请向系统管理员咨询。
                                                4. 如果所在组织要求使用安全连接 (https://)，请选中“使用安全连接”框。
                                                5. 选择“下一步”连接设备管理服务器。

                                                  注意：一旦您的设备与设备管理服务器建立了连接，Internet Explorer 窗口将显示欢迎信息。要继续执行，请关闭此窗口。

                                                                                延长电池寿命

                                                                                  您一定希望电池寿命越长越好，尤其是在旅行中。在正常情况下，一组电池可使用若干小时。下面都是一些很有用的提示。

                                                                                    • 尽量使用外部电源。
                                                                                      尽量使用交流电适配器将设备插入外部电源，尤其是遇到如下情形：
                                                                                      • 建立拨号连接（某些调制解调器耗电量很大）。
                                                                                      • 使用设备显示的背景光功能（如果有）。
                                                                                      • 连接到桌面计算机。
                                                                                      • 使用袖珍闪卡或其他外围设备。
                                                                                      • 让 LED 指示器闪光较长的时间。
                                                                                    • 设置设备在空闲时关闭。
                                                                                      在依靠电池供电的情况下，如果您三分钟内未碰键盘、笔针或鼠标，设备将自动关闭或挂起操作。缩短这一时间最大限度地延长电池寿命。
                                                                                    • 关闭没有必要的声音。
                                                                                      在默认情况下，设备生成的声音是用来响应各种事件的。例如，警告、约会和按下硬件按钮等。要优化电池的寿命，请关闭没有必要的声音。选择“开始”>“设置”>“控制面板”，然后选择“音量和声音”。

                                                                                    注意：电池电量不足时，任务栏将显示电池图标。

                                                                                      设置电源使用方案
                                                                                      1. 选择“开始”>“设置”>“控制面板”。
                                                                                      2. 打开“电源”图标，然后选择“方案”选项卡。
                                                                                      3. 选择合适的选项以进入低能状态。

                                                                                      注意：如果看不到“方案”选项卡，说明已启用被动的电源管理。有关被动电源管理的详细信息，请参阅设置设备在空闲时关闭。

                                                                                      选择的时间代表了系统切换至下一阶段的省电状态前必须经历的时间。例如，如果“将状态切换为系统空闲”的设置是“5 分钟后”，则系统将在处于不活动状态 5 分钟之后从“用户空闲”状态转换为“系统空闲”状态。

                                                                                      即便选中“从不”，系统仍在条件需要时进入低级的省电状态。例如，如果“将状态切换为挂起”的设置是“从不”，达到一定紧急程度的电池仍将导致系统挂起。

                                                                                      提示：较短的时间有助于节省电池电量。

                                                                                                  1. 选择“开始”>“设置”>“控制面板”。
                                                                                                  2. 打开“系统”图标。
                                                                                                  3. 在“内存”选项卡中，移动滑块为程序或存储分配更多的内存。
                                                                                                    如果没有足够的空间存储文件，请增加存储内存。
                                                                                                    如果移动设备运行缓慢，请尝试增加程序内存。
                                                                                                  注意：
                                                                                                  • 重置设备后便可以使用附加的存储或程序内存。如果仍有内存问题，请按制造商文档中的说明重置设备。
                                                                                                  • “常规”选项卡提供了与设备上的硬件和软件配置有关的常规信息。
                                                                                                  • 可分配给文件使用的最大 RAM 是 16 MB。

                                                                                                  设置设备在空闲时关闭

                                                                                                  ROM 中的程序

                                                                                                                  管理数字证书

                                                                                                                    证书是某些应用程序建立信任和安全通信的工具。证书由证书颁发机构签署和颁发，而且具有预先规定的有效期。Windows CE 可管理多种证书存储。

                                                                                                                    1. 选择“开始”>“设置”>“控制面板”。
                                                                                                                    2. 打开“证书”图标。
                                                                                                                    3. 在“存储”选项卡中，选择要查看的证书存储，或从下拉列表中选择要修改的证书。“可信颁发机构”存储列表显示了可信证书颁发机构的顶级证书。“我的证书”存储则包含了个人用于标识自己的证书。中间证书颁发机构有助于建立信任链，它保存在“其他颁发机构”存储中。
                                                                                                                    4. 要将证书或相关私钥添加到选定存储中，请选择“导入”。
                                                                                                                    5. 要查看选定证书的详细信息（如扩展名或截止日期），请选择“查看”。
                                                                                                                    6. 要从存储中删除选定证书，请选择“删除”。

                                                                                                                      管理存储设备

                                                                                                                        可用存储设备都按设备名列在下拉框中。要查看磁盘信息或执行存储操作，请从列表中选择设备。

                                                                                                                        • 要卸除选定存储设备上的所有分区，请选择“卸除”。
                                                                                                                        • 要格式化分区表，请选择“格式化”。如果装入了存储设备中的任意分区，则不能格式化分区表。

                                                                                                                        管理磁盘分区

                                                                                                                          选定存储设备上的可用分区将显示在列表框中。

                                                                                                                          • 要在存储设备上创建新的分区，请选择“新建”。
                                                                                                                          • 要从存储设备中删除分区，请选择“删除”。无法删除装入的分区。
                                                                                                                          • 要查看分区信息或执行高级分区功能（如格式化），请选择“属性”。
                                                                                                                          另请参阅
                                                                                                                          创建新分区
                                                                                                                          高级分区功能

                                                                                                                          创建新分区

                                                                                                                            1. 要创建新分区，请从列表中选择存储设备并选择“新建”。
                                                                                                                            2. 键入新分区的名称。
                                                                                                                            3. 键入新分区的大小（以扇区为单位），或选中“使用所有可用的磁盘空间”复选框。
                                                                                                                            4. 选择“确定”创建分区，或选择“X”取消。

                                                                                                                            高级分区功能

                                                                                                                              要查看高级分区功能，请选择某个分区，然后选择“属性”。

                                                                                                                              • 要装入某个卸除的分区，请选择“装入”。
                                                                                                                              • 要卸除某个已装入的分区，请选择“卸除”。
                                                                                                                              • 要格式化分区，请选择“格式化”。
                                                                                                                              • 要扫描并修复分区，请选择“扫描”。
                                                                                                                              • 要对分区进行磁盘碎片整理，请选择“整理碎片”。
                                                                                                                              注意：
                                                                                                                              • 根据分区文件系统的不同，“格式化”、“扫描”和“碎片整理”选项有可能无效。上述功能的行为随文件系统工具库的实现程度而异。

                                                                                                                                注意：要打开 Bluetooth，请选择“开始”>“设置”>“控制面板”，然后双击“Bluetooth”。


                                                                                                                                  扫描 Bluetooth 设备

                                                                                                                                    1. 选择“开始”>“设置”>“控制面板”，然后双击“Bluetooth”。
                                                                                                                                    2. 按下“扫描设备”按钮。这将启动查询进程。如果扫描结果发现类似的 Bluetooth 设备，设备将添加至“不受信任”列表框中。
                                                                                                                                    3. 然后，您可选择要与之建立信任关系的设备。有关针对信任关系对设备进行身份验证的详细信息，请参阅设备身份验证。
                                                                                                                                    另请参阅
                                                                                                                                    设备身份验证
                                                                                                                                    从受信任列表中删除设备
                                                                                                                                    激活设备

                                                                                                                                    设备身份验证

                                                                                                                                      1. 选择“开始”>“设置”>“控制面板”，然后双击“Bluetooth”。
                                                                                                                                      2. 在“不受信任”列表框中，单击要进行身份验证的设备。
                                                                                                                                      3. 单击“-->”按钮。如果设备需要授权，系统将弹出一个对话框，要求输入设备的 PIN（个人标识号）。成功授权后，设备将移至“受信任”列表框。
                                                                                                                                      另请参阅
                                                                                                                                      扫描 Bluetooth 设备
                                                                                                                                      从受信任列表中删除设备
                                                                                                                                      激活设备

                                                                                                                                      从受信任列表中删除设备

                                                                                                                                        1. 选择“开始”>“设置”>“控制面板”，然后双击“Bluetooth”。
                                                                                                                                        2. 在“受信任”列表框中，单击要删除的设备。
                                                                                                                                        3. 单击“<--”按钮使设备“不受信任”。
                                                                                                                                        4. 单击“DEL”按钮，删除该项并从缓存中清除。
                                                                                                                                        另请参阅

                                                                                                                                        激活设备

                                                                                                                                          对于每类设备，只能有一个设备可标记为“活动”并作为默认设备使用。

                                                                                                                                          1. 选择“开始”>“设置”>“控制面板”，然后双击“Bluetooth”。
                                                                                                                                          2. 在“受信任”列表框中，右键单击要改为“活动”状态的设备。
                                                                                                                                          3. 在弹出列表中，选择“活动”将设备标记为活动状态。
