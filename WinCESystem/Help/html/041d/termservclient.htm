Om Anslutning till fjärrskrivbord

Genom att använda Anslutning till fjärrskrivbord kan du logga in till en Windows Terminal Server och använda alla program som installerats på denna server. I stället för att exempelvis köra Microsoft Pocket Word kan du köra skrivbordsversionen av Microsoft Word.

    Tips

    Ansluta till en Terminal Server
    1. Välj Start > Program > Anslutning till fjärrskrivbord eller kör CETSC från DOS-prompten.
    2. I rutan Server anger eller väljer du ett Terminal Server-namn eller en TCP/IP-adress. Du kan också välja en server i listrutan.
    3. Välj Anslut.
    4. I fönstret Anslutning till fjärrskrivbord skriver du ditt användarnamn, lösenord och din domän (om det behövs) och väljer sedan OK.

      Koppla från utan att avsluta en session
      1. I fönstret Anslutning till fjärrskrivbord väljer du Start > Avsluta.
      2. Välj Koppla från > OK.

      Obs!   Om du tidigare kopplat bort från en Terminal Server utan att avsluta sessionen fortsätter Terminal Server att utföra alla processer som körs och Anslutning till fjärrskrivbord kan återansluta till samma session senare (om administratören har konfigurerat Anslutning till fjärrskrivbord så att den återansluter till sessioner som kopplats från).

        1. I fönstret Anslutning till fjärrskrivbord väljer du Start > Avsluta.
        2. Välj Logga ut > OK.

          Förbättra bildskärmens prestanda
          Du kan minska den tid det tar att rita om skärmen.
          1. I fönstret Terminal Server väljer du Start > Inställningar > Kontrollpanelen.
          2. Välj Bildskärm.  
          3. På vissa nyare Terminal Server-förekomster kanske du även måste markera Effekter på fliken Utseende.  
          4. Avmarkera Använd effekter för menyer och beskrivningar och Utjämna kanterna på skärmteckensnitt.

            Tips

            Förbättra prestanda med Internet Explorer
            Du kan minska den tid det tar att rita om webbsidor.
            1. I Internet Explorer väljer du Verktyg > Internet-alternativ.
            2. Välj fliken Avancerat.
            3. Avmarkera Använd mjuk rullning.

              Du kan använda den inbyggda hjälpen för att visa ytterligare kommandoradsalternativ genom att skriva:   CETSC /?
