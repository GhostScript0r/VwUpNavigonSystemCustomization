Windows Messenger-klient

    Med hjälp av klientprogrammet Windows CE Messenger i Windows CE kan du kommunicera i realtid genom att använda textbaserade meddelandefunktioner, VoIP osv.

    • Se vem som är online.
      Se när dina kollegor är online. Du kan också skicka din egen närvaroinformation till en lista över personer som du anger.
    • Tala i stället för att skriva
      Sluta skriv! Tala med en kollega var som helst i världen genom att använda enhetens mikrofon och högtalare. Windows Messenger stöder röstsamtal av hög kvalitet från Windows CE-enheten.
    • Kommunicera med SIP (Session Initiation Protocol) eller Hotmail-kontakter
      Windows Messenger stöder kommunikation med dina kollegor i företaget med en SIP-baserad kommunikationstjänst. Du kan även tala med dina Hotmail-kontakter genom att använda .NET Messenger.
