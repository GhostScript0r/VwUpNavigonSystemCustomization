Windows Messenger 用戶端

    Windows CE 中的 Windows CE Messenger 用戶端應用程式允許您使用文字訊息與 VoIP 等方式，與其他使用者即時通訊。

    • 看誰在線上
      看您的同事是否在線上。您也可以自行定義要顯示線上狀態的清單。
    • 以語音取代鍵盤輸入
      別再用鍵盤輸入了！您可以使用裝置上的麥克風與喇叭，與身處任何地方的同事談話。Windows Messenger 支援 Windows CE 裝置上高品質的語音通話。
    • 使用 Session Initiation Protocol (SIP) 或 Hotmail 連絡人進行通訊
      Windows Messenger 支援以 ISP 為基礎的通訊服務，可讓您與企業中的同事通訊。您也可以使用 .NET Messenger，與您的 Hotmail 連絡人通話。
